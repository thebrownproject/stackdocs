# Stackdocs Backend

FastAPI backend for Stackdocs MVP - document data extraction service.

## Tech Stack

- **Framework**: FastAPI
- **OCR**: Docling (bundled in monolith)
- **AI**: LangChain + Claude 3.5 Sonnet
- **Database**: Supabase PostgreSQL
- **Storage**: Supabase Storage
- **Async Processing**: FastAPI BackgroundTasks
